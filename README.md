# moneythat

## Project Description

moneythat is a simple web application that allows users to explore real-time cryptocurrency data in an easy and interactive way. The main idea behind the project is to make it easier to view and understand basic information about different cryptocurrencies without going through complex platforms.

The application fetches live data from the CoinGecko API and displays it in a clean and readable format. Users can quickly browse through different cryptocurrencies, check their current prices, and get a general overview of the market.

## Basic Functioning

When the application loads, it makes a request to the CoinGecko API and retrieves a list of cryptocurrencies along with their details. This data is then displayed dynamically on the webpage.

The user can interact with the data in a few simple ways:

* Search for a specific cryptocurrency using a search bar
* Filter the list based on certain conditions such as price or market-related values
* Sort the data in different orders like ascending or descending

All these interactions update the displayed data in real time without reloading the page.

The project focuses on using JavaScript to handle data fetching and manipulation, along with basic HTML and CSS for structure and styling.
